Button Components


Los botones en Bootstrap son elementos interactivos usados para ejecutar acciones, enviar formularios o navegar entre páginas. Bootstrap facilita su uso con clases predefinidas que aplican estilos, colores y tamaños consistentes.


Bootstrap incluye varias variantes de botones, cada una con su propio propósito semántico, y algunos extras para mayor control.

Placeholders


on elementos visuales temporales que se utilizan para simular el contenido que aún no se ha cargado. Son muy útiles cuando una página está esperando datos de una API o base de datos, y quieres mostrar al usuario una vista previa del diseño mientras la información real se carga.








Popovers


son pequeñas ventanas flotantes que aparecen cuando el usuario hace clic o pasa el cursor sobre un elemento. Se usan para mostrar información adicional sin tener que cargar una nueva página o cambiar el diseño principal. Son muy parecidos a los tooltips, pero los popovers pueden contener más contenido, como títulos, párrafos o botones.








Progress


El componente Progress se usa para mostrar el progreso o avance de una tarea, proceso o carga de datos. Es una barra visual que puede llenarse parcialmente según el porcentaje completado.













Scrollspy


El componente Scrollspy de Bootstrap permite que un elemento de navegación (navbar o lista) marque automáticamente la sección activa de una página mientras el usuario se desplaza. En otras palabras, resalta el enlace activo según la posición del scroll en la página.

Tooltips


Los Tooltips son pequeños cuadros de texto que aparecen cuando el usuario pasa el cursor o toca un elemento. Se usan para mostrar información adicional o aclaraciones sin ocupar espacio en la página. Por ejemplo: mostrar una explicación cuando el usuario coloca el ratón sobre un botón o un enlace.
